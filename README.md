## 🌌 Overview
**A.K** ek high-performance infinite runner hai jo browser mein 3D space create karta hai. Yeh normal games se alag hai kyunki isme aapke haath nahi, aapki **Voice** command deti hai.

---

## 🚀 Future-Gen Features
- 🎤 **Voice Sync:** Web Speech API ka use karke real-time voice recognition.
- 📐 **3D Grid World:** Pure CSS transforms se banaya gaya 80s Cyberpunk grid.
- 📱 **Adaptive UI:** Mobile par touch aur desktop par voice/keys ka combo.
- ⚡ **Neon HUD:** Glassmorphic heads-up display jo score aur energy track karta hai.
